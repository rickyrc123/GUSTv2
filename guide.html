GUSTv2 Setup Guide

Introduction

Welcome to the GUSTv2 setup guide. This document will walk you through the steps to set up and configure GUSTv2.

Prerequisites

  • Python 3.8 or higher installed
  • Basic understanding of Python and command-line tools
  • Required dependencies installed (see below)

Installation Steps

  1. Download the GUSTv2 source code from the official repository.
  2. Open a terminal and navigate to the project directory.
  3. Run pip install -r requirements.txt to install dependencies.
  4. Execute python main.py to start the application.

Troubleshooting

If you encounter any issues, please refer to the FAQ or contact support.
