API Documentation

Endpoints in fast_api_test_server.py

GET /test_data/get_generated_data

Generates random test data for drones.

Response: JSON payload containing randomly generated drone data.

GET /drones

Retrieves a list of all drones currently in the system.

Response: JSON object with a list of drones.

POST /drones/create

Creates a new drone in the system.

Request Body: JSON object containing drone details.

Field Type Description
name string Name of the drone
model string Model of the drone
status string Status of the drone

Response: Status message indicating success or failure.

GET /drones/{drone_name}

Retrieves details of a specific drone by its name.

Path Parameter: drone_name - Name of the drone.

Field Type Description
drone_name string Name of the drone

Response: JSON object with drone details.

POST /drones/{drone_name}/delete

Deletes a specific drone by its name.

Path Parameter: drone_name - Name of the drone.

Field Type Description
drone_name string Name of the drone

Response: Status message indicating success or failure.

POST /drones/{drone_name}/update_position

Updates the last known position of a specific drone.

Path Parameter: drone_name - Name of the drone.

Field Type Description
drone_name string Name of the drone

Response: Status message indicating success or failure.
